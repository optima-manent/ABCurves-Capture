# Security

Please report security or privacy issues through GitHub's
[private vulnerability reporting form](https://github.com/optima-manent/ABCurves-Capture/security/advisories/new)
instead of opening a public issue.

Include the affected version, a short description, and reproduction steps if
you have them. In particular, please report anything that could expose data from
the wrong USB device, weaken the normal-user/elevated-helper boundary, escape a
session or ZIP path, or corrupt a completed archive.

Do not attach a real participant session to an issue. A small synthetic example
is much easier and safer to work with.
